# Bioinformatics
This repo is used for class Bioinformatics in spring of 2026, including homework, notes, etc.
## 学习计划
1. 选择并学习这门课程,主要有三个目的: \
加强基本的计算机能力,包括但不限于programing水平,markdown熟练使用,常用网站和平台如github的运用,能够基本"入门"计算领域; \
掌握生物信息学的基本方法,并了解其原理; \
能够在自己的研究和项目中真正运用到这些方法,或有能力学习拓展的方法并应用. 
2. 个人学习计划如下: \
   2.1 课前预习: 根据教师提供的课程信息(https://mkdocs.ncrnalab.org/courses/bioinfo/ ),了解本周上课要讲的内容. 对于特别感兴趣或先前不了解的部分,画出重点并带着这些问题去课堂. 主要关注的内容是: slides, Syllabus 对应部分的内容. \
   2.2 课堂听讲: 到课并听讲,记录课堂笔记. 对不理解的问题及时与老师和助教交流解决,当日事当日毕. \
   2.3 课后复习: 首先完成作业. 然后再复习上课内容. 主要关注Tutorial对应部分的内容. 在有余力的情况下,阅读学习教材和额外的课程材料. 记得预习新一周的内容. \
   2.4 其他: 准备好课本. 多与同班同学讨论,点滴学习,把知识用到实践中去.
